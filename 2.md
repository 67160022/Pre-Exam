# 2. HTML5 และ Semantic Markup

**ให้เขียนรหัส HTML5 ส าหรับหน้าแสดงบทความข่าวสาร (Blog Post) โดยต้องประกอบด้วย: header ที่มี logo และ navigation menu, main content ที่มีบทความย่อยหลายข้อ, sidebar
สำหรับหมวดหมู่, และ footer ที่มี social links ใช้ semantic elements อย่างถูกต้อง (เช่น `<header>`, `<nav>`, `<article>`, `<section>`, `<aside>`, `<footer>`)**

**ตอบ**

```
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>
  <body>
    <header>
      <h1>Blog Post</h1>

      <nav>
        <ul>
          <li><a href="#">หน้าแรก</a></li>
          <li><a href="#">บทความ</a></li>
          <li><a href="#">เกี่ยวกับ</a></li>
          <li><a href="#">ติดต่อ</a></li>
        </ul>
      </nav>
    </header>

    <main>
      <section>
        <h2>บทความล่าสุด</h2>

        <article>
          <h3>หัวข้อบทความที่ 1</h3>
          <p>เนื้อหาบทความสั้น ๆ เกี่ยวกับเรื่องนี้...</p>
        </article>

        <article>
          <h3>หัวข้อบทความที่ 2</h3>
          <p>การบรรยายของหัวข้อนี้ ...</p>
        </article>

        <article>
          <h3>หัวข้อบทความที่ 3</h3>
          <p>เนื้อหาของหัวข้อนี้ ...</p>
        </article>
      </section>
    </main>

    <aside>
      <h2>หมวดหมู่</h2>
      <ul>
        <li><a href="#">กีฬา</a></li>
        <li><a href="#">การท่องเที่ยว</a></li>
        <li><a href="#">การศึกษา</a></li>
      </ul>
    </aside>

    <footer>
      <p>© 2026 MyBlog</p>
      <p>ติดตามเรา:</p>
      <a href="#">Facebook</a> |
      <a href="#">Instagram</a>
    </footer>
  </body>
</html>
s
```
